# Privacy Policy — BulkyURLs

**Last updated: 2026-03-11**

## Data Collection

BulkyURLs does not collect, transmit, or share any personal data.

## Data Storage

The only data stored by this extension is your configuration (key bindings, colors, blocked-site patterns) in Chrome's `chrome.storage.local` API. This data:

- Stays entirely on your local device
- Is never sent to any server or third party
- Can be cleared at any time by removing the extension

## Permissions

| Permission | Why it's needed |
|---|---|
| `activeTab` | Read the current tab's URL when the popup is open |
| `tabs` | Open selected links in new tabs |
| `storage` | Save your settings locally |
| `contextMenus` | Add right-click menu items for adding/collecting links |
| `http://*/*`, `https://*/*` | Inject the drag-select content script into web pages |
